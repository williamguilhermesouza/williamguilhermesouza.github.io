Quem sou

Desenvolvedor Fullstack, apaixonado pela área de desenvolvimento e envolvido no estudo de tecnologia e área de TI desde 2003. Atualmente trabalhando na área de desenvolvimento como freelancer, especializado na idealização, design, prototipagem e implementação de sistemas web, desde o frontend até o backend e infraestrutura envolvida.

O que faço

Atuo buscando prover todos os elementos necessários para as soluções de TI idealizadas pelos clientes, com o fim de atender todos os requisitos levantados e idealização da interface com maior fidelidade possivel. Abaixo estão as principais tecnologias e ferramentas que utilizo para o desenvolvimento das soluções de TI:

Clean Code

Busco o desenvolvimento com código limpo, legível e de fácil manutenção, obedecendo todas as determinações de documentação e Boas Práticas.

JavaScript ES 6

Experiência com desenvolvimento JavaScript utilizando as últimas features da linguagem.

Typescript

Utilização do superset/linguagem tipada Typescript, buscando dar maior segurança e estabilidade as aplicações.

React e React-Native

Conhecimento nas tecnologias React e React-Native, para sistemas frontend e desenvolvimento de aplicativos para smartphones.
